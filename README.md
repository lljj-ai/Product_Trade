# 商品管理以及交易系统
## 前端技术
- ajax+jQuery+bootstrap+jsp
## 后端技术
- servlet+C3P0连接池
## 数据库技术
- mysql
## 其余技术
- 图表（echarts）
- 分页（page.js）
## 系统功能
- 登录
- 注册
- 所有表单提交的验证
- 动态验证码
- 记住秘密（session存储，七天）
- 商品交易表的列表显示（柱状图、折线图、饼图的方式显示）
- 商品信息表的显示、添加、删除、修改功能（radio选择进行修改和删除），且修改时数据回显，id只读权限
- 买家信息的显示、查询（id查询）添加、修改、删除（每条记录对应一个删除和修改按钮）
